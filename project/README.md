> Спеціальність: Системний аналіз та управління, КПІ, 3 курс, весна 2026

---

## Що це?

Навичка (skill) для AI-агента ClawdBot, яка дозволяє вести домашній бюджет через Telegram.

Користувач пише природною мовою:

```
«Витратив 850 грн на продукти»
«Заробив $500 фріланс»
«Скільки витратив цього місяця?»
```

Агент розпізнає намір, зберігає дані у PostgreSQL і відповідає підтвердженням або звітом.

## Швидкий старт для розробника

```bash
# 1. Клонуємо репозиторій
git clone <repo-url>
cd project

# 2. Переходимо в папку імплементації
cd 10_Implementation

# 3. Встановлюємо залежності (після створення package.json)
npm install

# 4. Запускаємо тести
npm test
```
